Utilizei para o desenvolvimento da API o framework Spring, pois o mesmo possibilita um desenvolvimento mais rápido e prático. Utilizei o Springboot para ficar mais simplificado a parte de configurações, utilizei um banco de dados em memória, H2, para conseguir simular requisições reais em um servidor além de poder utilizar o SpringData JPA que simplifica bastante a parte de Banco de Dados. Procurei dividir cada cada em pastas especificas e abstrair em utils algo em comum para os modulos. Coloquei como dependencia no POM o Docker sendo possiivel 'buildar' uma imagem a partir de comando do maven.

Para o desnevolvimento do Front utilizei o Angular pois acho ele mais intuitivo e mais próximo dos padrões de desenvolvimento backend tornando o desenvolvimento um pouco mais prático. procurei componentizar cada parte do sistema para possíveis re-uso de código. Utilizei os componentes da primeNG pois eles já vem prontos para uso com pouca configuração, tornando o desenvolvimento um pouco mais produtivo, também coloquei docker com o nginx para proxy reverso para a porta 80.

Docker, estou utilizando o Docker compose, assim para melhorar a manipulação   de todos os containers do projeto.
